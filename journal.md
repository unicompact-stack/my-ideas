- Создана папка `полуавтоматизация-объявлений/` — помощник для создания объявлений по одному слову. Папки: `заготовки/`, `промты-для-картинок/`, `примеры-объявлений/`.
- Создана структура `agents/` — AI-агенты для поиска конкурентов и анализа через OpenRouter. Ключи хранятся в `.env` (не в GitHub). Файлы: `search_agent.py`, `analyze_agent.py`, `error_checker.py`, `requirements.txt`, `README.md`. Также добавлены `.gitignore` и `research/конкуренты.md`.
  - Поиск работает через `googlesearch-python` (бесплатно, без API-ключей, ~10–15 запросов/час).
  - Анализ идёт через OpenRouter API (модель `deepseek/deepseek-chat` по умолчанию).
  - `error_checker.py` проверяет наличие `.env`, всех ключей и установленных библиотек перед запуском.

## 🛠️ Рабочие инструкции

### Как добавить новую папку
1. Сказать Perplexity: «В репозитории my-ideas создай папку [название] со структурой: MASTER.md, стратегия.md, chunks.md, инструкции.md».
2. Заполнить файлы по шаблону.
3. Записать в `journal.md` что добавили и зачем.

### Как запустить агентов
1. Создать `.env` в корне с ключом `OPENROUTER_API_KEY`.
2. `pip install -r agents/requirements.txt`
3. `python agents/error_checker.py` — проверка
4. `python agents/search_agent.py "запрос"` — поиск
5. `python agents/analyze_agent.py` — анализ
