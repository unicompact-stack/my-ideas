# Моя цель: проверять идеи на жизнеспособность

**Главная цель:** не тратить время и деньги на идеи, которые не работают. Быстро проверять гипотезы и отсеивать слабые.

**Что я уже умею:**
- Формулировать идеи.
- Работать с Perplexity, Cline, GitHub.
- Делать прототипы.

**Правила:**
- Каждая идея проверяется по 3 критериям:
  1. Есть ли спрос? (кто будет платить)
  2. Можно ли сделать быстро? (за 1-2 дня)
  3. Есть ли рынок? (конкуренты, аналоги)
- Если идея не проходит 2 из 3 критериев — отказ.
- Результаты проверок сохранять в `инструкции.md`.
