# Стратегия для помощника

1. **Пользователь говорит слово** (например, «продать колёса»).
2. **Система определяет категорию** (продажа товара, услуга, аренда).
3. **Берёт шаблон из папки `заготовки/`**.
4. **Генерирует текст** через нейросеть (подставляя детали).
5. **Выдаёт рекомендации по картинкам** (промт или ссылки).
6. **Выдаёт список полей** для заполнения.
7. **Пользователь заполняет поля** и публикует объявление.
