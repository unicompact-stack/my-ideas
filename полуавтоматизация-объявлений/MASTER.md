# Полуавтоматический помощник для объявлений

**Главная цель:** по одному слову (например, «продать колёса») выдавать:
- Готовый текст объявления.
- Рекомендации по картинкам (промт для генерации или ссылки на похожие).
- Список полей для заполнения (цена, контакты, условия).

**Что я уже умею:**
- Работать с нейросетями (Perplexity, Cline).
- Генерировать тексты и промты.

**Правила:**
- Текст должен быть адаптирован под площадку (Авито, ВК, Telegram).
- Картинки — либо готовые промты для генерации, либо ссылки на примеры.
- Поля для заполнения — всегда список (цена, город, контакты).
