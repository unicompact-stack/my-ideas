# Разбивка на шаги

1. Создать шаблоны для разных категорий (продажа, услуги, аренда).
2. Написать промты для генерации текста через Perplexity.
3. Написать промты для генерации картинок (через Midjourney или другой AI).
4. Добавить список полей для каждой категории.
5. Сделать интерфейс (или просто инструкцию) для пользователя.
6. Протестировать на 3 разных объявлениях.
