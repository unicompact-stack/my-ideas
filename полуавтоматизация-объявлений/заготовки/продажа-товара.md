# Шаблон для продажи товара

**Структура текста:**
1. Заголовок: [Товар] в отличном состоянии!
2. Описание: [краткое описание, состояние, пробег/гарантия]
3. Цена: [цена] руб.
4. Контакты: [телефон, Telegram]

**Поля для заполнения:**
- Название товара
- Состояние (новый/б/у)
- Цена
- Город
- Контакты

**Рекомендации по картинкам:**
- Промт для генерации: «[товар] на белом фоне, студийное освещение».
- Или ссылки на похожие фото.
