# Шаблон для аренды

**Структура текста:**
1. Заголовок: Сдам [объект] — [ключевая характеристика]!
2. Описание: [площадь/размер, условия, что включено]
3. Цена: [цена] руб./мес.
4. Контакты: [телефон, Telegram]

**Поля для заполнения:**
- Объект аренды
- Площадь / характеристики
- Цена
- Город, район
- Контакты
- Условия (залог, коммуналка и т.д.)

**Рекомендации по картинкам:**
- Фото объекта при хорошем освещении.
- Промт: «[объект] интерьер, светлая комната, уютно».
