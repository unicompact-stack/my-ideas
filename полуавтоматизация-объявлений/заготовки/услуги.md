# Шаблон для услуг

**Структура текста:**
1. Заголовок: [Услуга] — быстро и качественно!
2. Описание: [что делаю, опыт, результат]
3. Цена: от [цена] руб.
4. Контакты: [телефон, Telegram]

**Поля для заполнения:**
- Название услуги
- Опыт (лет/кейсы)
- Цена
- Город / онлайн
- Контакты

**Рекомендации по картинкам:**
- Фото результата работы.
- Промт: «[услуга], профессиональный стиль, светлый фон».
