# strategy.md — План проекта

## Что мы строим
Простой бот, который:
1. Генерирует текст через ИИ.
2. Публикует текст в ВК (через код).
3. Публикует картинки в Pinterest (через Browser Agent).

## Как это работает
- ВК: пишем код на Python, отправляем запрос через VK API.
- Pinterest: просим Perplexity открыть браузер и нажать кнопки.

## Что нам нужно
- Python (для кода)
- VK API (токен из ВК)
- Perplexity с Browser Agent (для Pinterest)
- GitHub (храним всё здесь)

## Где хранить
- `.env` — пароли и токены (НИКОГДА не пушить в GitHub!)
- `posts/` — готовые тексты и картинки
- `modules/` — код для ВК и Pinterest
