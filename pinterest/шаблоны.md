# шаблоны.md — Шаблоны для пинов Pinterest

## Структура пина
- **Картинка**: вертикальная, соотношение 2:3 (например, 1000×1500 px)
- **Заголовок**: до 100 символов, включай ключевые слова
- **Описание**: 200-500 символов, описывай пользу
- **Доска**: выбери нужную доску

## Шаблон описания
```
[Что изображено / главная польза]
[Дополнительные детали]
[Призыв перейти на сайт или сохранить]
```
