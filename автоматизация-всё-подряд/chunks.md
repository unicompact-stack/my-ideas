# Разбивка на шаги

## Для простых идей (напоминалки)
1. Создать файл в `простые-напоминалки/`.
2. Написать код (через Cline или руками).
3. Проверить работу.
4. Записать результат в `инструкции.md`.

## Для средних идей (боты)
1. Создать файл в `средние-боты/`.
2. Написать ТЗ (что должен делать).
3. Написать код (через Cline).
4. Протестировать, записать вывод.

## Для сложных идей (AI-агенты)
1. Создать файл в `сложные-агенты/`.
2. Провести глубокое исследование через Perplexity.
3. Оценить стоимость (API, сервера).
4. Принять решение.
