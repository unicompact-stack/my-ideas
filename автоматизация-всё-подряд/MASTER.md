# Моя цель: собирать и проверять идеи по автоматизации

**Главная цель:** найти работающие идеи для автоматизации (напоминалки, боты, AI-агенты) и проверять их на жизнеспособность.

**Что я уже умею:**
- Работать с VK API, Telegram API.
- Писать простые боты.
- Использовать Perplexity и Cline.

**Правила:**
- Идея должна быть реализуема с помощью AI (без сложного программирования).
- Для каждой идеи — минимум 1 конкурент/аналог.
- Если идея требует сложных ресурсов (дорогой API, сервера) — оценить стоимость.
- Работающие идеи — в `инструкции.md`, провальные — туда же с пометкой.
