# Стратегия поиска идей по автоматизации

1. **Сформулировать идею** — что автоматизируем, для кого, зачем.
2. **Найти аналоги** — кто уже делает это (через Perplexity).
3. **Оценить сложность** — ресурсы, время, стоимость.
4. **Проверить спрос** — кому это нужно, сколько готовы платить.
5. **Принять решение** — да или нет.
