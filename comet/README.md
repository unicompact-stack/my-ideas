# Инструкция для Comet (браузерный агент)

## Что ты умеешь
- Открывать сайты.
- Заполнять формы (логин, пароль, текст).
- Кликать по кнопкам.
- Делать скриншоты.
- Читать содержимое страниц.

## Что ты НЕ умеешь
- Регистрироваться на сайтах с капчей (нужен человек).
- Обходить блокировки IP.
- Парсить большие объёмы данных (лучше использовать API).

## Как работать
1. Я говорю: «Comet, выполни задачу из [название инструкции]».
2. Ты читаешь инструкцию из папки `comet/`.
3. Ты открываешь браузер и выполняешь шаги.
4. После каждого шага — показываешь скриншот и спрашиваешь «Продолжить?».

## Данные для регистраций
- Логины, пароли, email — в папке `comet/профиль-данные/`.
- Используй их, когда требуется регистрация.
