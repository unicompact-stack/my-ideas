# Инструкция для Comet: Поиск конкурентов на Авито

## Цель
Найти объявления конкурентов в категории «услуги».

## Шаги
1. Открой avito.ru.
2. В поиске введи «автопостер» или «SMM услуги».
3. Сделай скриншот результатов.
4. Открой первые 3 объявления.
5. Для каждого: скопируй заголовок, цену, текст.
6. Сохрани результат в файл `research/конкуренты-авито.md`.

## Формат сохранения
```markdown
# Конкуренты на Авито

## Объявление 1
- Заголовок: [текст]
- Цена: [текст]
- Текст: [текст]

## Объявление 2
- Заголовок: [текст]
- Цена: [текст]
- Текст: [текст]

## Объявление 3
- Заголовок: [текст]
- Цена: [текст]
- Текст: [текст]
```

## Если что-то пошло не так
- Сделай скриншот.
- Остановись и скажи: «Ошибка: [описание]».
